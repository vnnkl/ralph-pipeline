  <files>lib/time-budget.cjs</files>
  <action>
Create lib/time-budget.cjs with four exported functions. Follow the immutable coding style (create new objects, never mutate). Follow the code example from 05-RESEARCH.md closely.

**cmdTimeBudgetStart(cwd, hours):**
- Validate hours is positive number, else error('Hours must be a positive number', 'INVALID_HOURS')
- Load config via loadConfig(cwd)
- Compute expiresAt = Date.now() + (parseFloat(hours) * 3600000)
- Create new config object with spread: { ...config, time_budget_hours: parseFloat(hours), time_budget_expires: expiresAt }
- Save via saveConfig(cwd, updatedConfig)
- Output: { started: true, hours, expires_at: new Date(expiresAt).toISOString() }

**cmdTimeBudgetCheck(cwd):**
- Load config
- If no time_budget_expires: output({ has_budget: false, expired: false })
- Else: compute remaining = config.time_budget_expires - Date.now()
- expired = remaining <= 0
- Format remaining_display: "Xh Ym remaining" or "Budget expired"
- Output: { has_budget: true, expired, remaining_ms, remaining_display, budget_hours, expires_at }

**cmdTimeBudgetRecordBead(cwd, durationMs):**
- Validate durationMs, else error('Duration in ms required', 'INVALID_DURATION')
- Load config, get prevAvg and prevCount
- Compute weighted running average: newAvg = prevCount === 0 ? duration : Math.round((prevAvg * prevCount + duration) / newCount)
- Save updated config with new avg and count
- Output: { recorded: true, duration_ms, avg_bead_duration_ms, total_beads_executed }

**cmdTimeBudgetEstimate(cwd):**
- Load config
- avgBead = config.avg_bead_duration_ms or default 20 * 60 * 1000 (20 min)
- remaining = time_budget_expires ? Math.max(0, expires - now) : null
- estimatedBeads = remaining !== null ? Math.floor(remaining / avgBead) : null
- Output: { avg_bead_duration_ms, avg_bead_duration_display, total_beads_executed, remaining_ms, estimated_beads_remaining, is_first_run }

All functions must use immutable patterns -- create new config objects with spread, never mutate the loaded config.

2. Add to the switch/case router:
```javascript
case 'time-budget': {
  const sub = args[1];
  if (!sub) error('Usage: time-budget <start|check|record-bead|estimate> [args]', 'INVALID_ARGS');
  switch (sub) {
    case 'start':
      timeBudget.cmdTimeBudgetStart(cwd, args[2]);
      break;
    case 'check':
      timeBudget.cmdTimeBudgetCheck(cwd);
      break;
    case 'record-bead':
      timeBudget.cmdTimeBudgetRecordBead(cwd, args[2]);
      break;
    case 'estimate':
      timeBudget.cmdTimeBudgetEstimate(cwd);
      break;
    default:
      error('Unknown time-budget subcommand: ' + sub, 'UNKNOWN_SUBCOMMAND');
  }
  break;
}
```
