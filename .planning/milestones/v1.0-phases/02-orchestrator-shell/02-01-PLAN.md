**scanPipelinePhases(cwd):**
- Read .planning/pipeline/ directory for output files named `{phase.name}.md`
- For each PIPELINE_PHASES entry: check if output file exists, extract frontmatter, check `completed === true`
- Use safeReadFile + extractFrontmatter from existing libs (don't hand-roll)
- Return array of { ...phase, outputExists: boolean, completed: boolean }

**detectPosition(cwd, statePhase):**
- Call scanPipelinePhases(cwd) to get file scan results
- Find first phase where completed === false (this is the "file scan position")
- If all completed: return { phase: null, pipeline_complete: true }
- Compare statePhase (integer) with file scan position
- If match: return { phase: fileScanPhase, mismatch: false }
- If mismatch: return { phase: fileScanPhase, mismatch: true, corrected_from: statePhase }
- Per locked decision: trust file scan on mismatch (more recent truth)

**fillTemplate(templateContent, variables):**
- For each [key, value] in variables: replace `{{KEY}}` globally with String(value)
- After all substitutions: check for remaining `{{[A-Z_]+}}` patterns
- If unresolved remain: throw new Error('Unresolved template variables: ' + unresolved.join(', '))
- Return filled string

**excerptFile(filePath, maxLines = 10):**
- Read file with safeReadFile; return null if file doesn't exist
- Strip YAML frontmatter (if present) using regex: /^---\n[\s\S]*?\n---\n*/
- Split remaining content into lines, take first maxLines non-empty lines
- Return joined string

**CLI commands (cmdScanPhases, cmdExcerpt):**

cmdScanPhases(cwd, raw):

- Output as JSON via output()

cmdExcerpt(cwd, filePath, maxLines, raw):
- Validate filePath exists
- Call excerptFile(filePath, maxLines)
- Output { file: filePath, lines: maxLines, excerpt } via output()
