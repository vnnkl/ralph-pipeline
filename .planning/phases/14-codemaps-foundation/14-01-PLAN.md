**Pure functions (testable, no side effects):**

`checkFreshness(cwd)`:
- Resolve dir: `path.join(cwd, CODEMAP_DIR)`
- Check all 7 files exist via `fs.statSync()` in try/catch
- If any missing: return `{exists: false, fresh: false}`
- Compute oldest mtime: `Math.min(...mtimes)`
- Compute age: `Date.now() - oldestMtime`
- Return `{exists: true, fresh: age < STALENESS_THRESHOLD_MS}`

`getCodemapPaths(cwd)`:
- Map CODEMAP_FILES to absolute paths: `path.join(cwd, CODEMAP_DIR, f)`
- Return `{paths: absolutePaths}`

`getCodemapAge(cwd)`:
- Resolve dir, stat all 7 files
- If any missing: return `{exists: false, age_hours: null}`
- Find oldest and youngest by mtime
- Compute age in hours: `(Date.now() - oldestMtime) / (1000 * 60 * 60)`, rounded to 1 decimal
- Return `{exists: true, age_hours, oldest_file, youngest_file}`

**CLI wrappers:**

```javascript
function cmdCodemapCheck(cwd) { output(checkFreshness(cwd)); }
function cmdCodemapPaths(cwd) { output(getCodemapPaths(cwd)); }
function cmdCodemapAge(cwd) { output(getCodemapAge(cwd)); }
```
