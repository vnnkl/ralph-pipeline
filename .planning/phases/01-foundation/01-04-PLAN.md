**cmdInitPipeline(cwd, raw):**
The main init for when the skill is first invoked. Returns everything the orchestrator needs to decide what to do next:
```javascript
{
  // Config
  mode: config.mode,              // 'normal' | 'yolo'
  depth: config.depth,            // 'quick' | 'standard' | 'comprehensive'
  commit_docs: config.commit_docs,
  auto_advance: config.auto_advance,
  time_budget: config.time_budget,
  ide: config.ide,

  // State
  current_phase: /* extracted from STATE.md body */,
  phase_name: /* looked up from phase directory name */,
  status: /* extracted from STATE.md */,

  // Phase info
  phase_dir: /* from findPhaseInternal() */,
  phase_count: /* count of phase directories in .planning/phases/ */,

  // File existence checks
  state_exists: pathExistsInternal(cwd, '.planning/STATE.md'),
  roadmap_exists: pathExistsInternal(cwd, '.planning/ROADMAP.md'),
  project_exists: pathExistsInternal(cwd, '.planning/PROJECT.md'),
  config_exists: pathExistsInternal(cwd, '.planning/config.json'),
  reference_exists: pathExistsInternal(cwd, '.reference/get-shit-done'),

  // Preflight cache (avoid re-running if recent)
  preflight_passed: /* check if .planning/.preflight-cache.json exists and is < 1 hour old */,
}
```

Read STATE.md via safeReadFile, extract fields via stateExtractField. Look up phase directory via findPhaseInternal. Count phase dirs by reading .planning/phases/ directory.

For preflight cache: check `.planning/.preflight-cache.json` — if exists and `timestamp` field is within last hour, use cached `passed` value. Otherwise return null (orchestrator should run preflight).

**cmdInitPhase(cwd, phaseIdentifier, raw):**
Init for a specific phase context (used by phase subagents):
```javascript
{
  // Everything from pipeline init
  ...pipelineInit,

  // Phase-specific
  phase_number: N,
  phase_dir: '/absolute/path',
  has_context: /* 01-CONTEXT.md exists in phase dir */,
  has_research: /* 01-RESEARCH.md exists in phase dir */,
  has_plans: /* count of *-PLAN.md files */,
  has_summaries: /* count of *-SUMMARY.md files */,
  plan_files: /* array of PLAN.md filenames */,
  summary_files: /* array of SUMMARY.md filenames */,
}
```

**Wire into ralph-tools.cjs:**
Add `const init = require('./lib/init.cjs');` at top.
Wire init command with subcommands:
- `init pipeline` -> init.cmdInitPipeline(cwd, raw)
- `init phase {N}` -> init.cmdInitPhase(cwd, args[1], raw)
- `init` (no subcommand) -> default to `init pipeline`
